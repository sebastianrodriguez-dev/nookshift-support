# Security

NookShift's public support site is a static GitHub Pages site. It does not run server-side code, accept form submissions, set cookies, load third-party scripts, or store visitor data.

To report a security or privacy concern, email:

`nookshift.support.gz7il@passmail.net`

Please do not open a public issue containing passwords, Apple Account information, payment details, recovery codes, verification codes, private screenshots, or other sensitive information.

Include the affected page, a clear description of the issue, and reproduction steps that do not expose private data.
